# Bienvenido a INVENTO

**INVENTO** es un sistema de gestión de inventarios diseñado específicamente para pequeñas y medianas empresas (PYMEs) que buscan digitalizar su control de stock y eliminar el uso de hojas de cálculo manuales.

Desarrollado como proyecto formativo para el programa **Técnico en Programación de Software - SENA**, este sistema proporciona una solución práctica y accesible para empresas que están dando sus primeros pasos en la digitalización de sus procesos.

## ¿Por qué INVENTO?

<Note>
  **INVENTO** elimina los errores humanos comunes en hojas de cálculo y proporciona un sistema centralizado para que todo tu equipo pueda acceder y actualizar el inventario de manera segura.
</Note>

Las pequeñas y medianas empresas a menudo enfrentan desafíos al gestionar su inventario:

- **Errores manuales**: Las hojas de cálculo son propensas a errores de entrada de datos
- **Falta de control**: Múltiples personas editando archivos locales causan inconsistencias
- **Sin trazabilidad**: Difícil rastrear quién hizo qué cambio y cuándo
- **Acceso limitado**: Compartir archivos por correo o USB es ineficiente

INVENTO resuelve estos problemas proporcionando una interfaz web centralizada, accesible desde cualquier navegador, con control de acceso por roles y trazabilidad completa de todos los movimientos.

## Tecnología simple y accesible

El sistema está construido con tecnologías estándar y ampliamente soportadas:

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: PHP (sin frameworks complejos)
- **Base de datos**: MySQL
- **Servidor**: Apache (incluido en XAMPP)

<Tip>
  Esta combinación de tecnologías hace que INVENTO sea fácil de instalar y mantener, incluso para pequeños negocios sin un departamento de IT dedicado.
</Tip>
